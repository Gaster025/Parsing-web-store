# Parsing web-store

Программа на Python, которая будет скрапить информацию о
названии, цене и описании товаров с веб-страницы магазина ´"OZON"´ 
и сохранять её в файл CSV.

## Стек

- Python > 3.10.12
	- requests > 2.31.0
	- BeautifulSoup4 > 4.12.2
	- Selenium > 4.16.0
	- time 
	- CSV
- VScode
